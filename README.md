# rff-setup
Setup script for RFF tools.

## Usage

```shell
$ ./setup.sh
```

## Features

* Check whether Node.js is installed.
* Check whether Ruby is installed.
* Install yo.
* Install Grunt.
* Install Bower.
* Install Sass.
* Install rff-doctor and run it.

## Requirement

* Mac OS X
* Git Bash on Windows
